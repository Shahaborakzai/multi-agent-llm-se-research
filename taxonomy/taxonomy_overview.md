# Agent Role Taxonomy Overview

## Derived from 19 SOTA Papers (2023–2025)

## The 7-Layer Taxonomy

| Layer | Purpose | Representative Roles | Example Systems |
|-------|---------|---------------------|-----------------|
| 1. Orchestration | Manage coordination | Manager, Supervisor | MetaGPT, Magentic-One |
| 2. Analysis & Planning | Define requirements | PM, Planner | MetaGPT, MapCoder |
| 3. Design | Specify architecture | Architect, Designer | MetaGPT, PotPie |
| 4. Implementation | Create artifacts | Coder, Implementer | WorldCoder, Tree-of-Code |
| 5. Quality Assurance | Verify outputs | Reviewer, Tester | CodeAgent, PotPie |
| 6. Knowledge & Docs | Maintain repository | KG Curator, Writer | PotPie, MetaGPT |
| 7. Operations | Execute & monitor | Executor, Interface | Tree-of-Code |

## Key Finding
QA layer dominates (9/19 papers) — strong trend toward
autonomous verification and self-correction.
